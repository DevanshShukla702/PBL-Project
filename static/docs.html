POST /predict-route-eta Compute multi-horizon routes & ETAs

Receives geographical endpoints and calculates k=3 distinctly contextualized physical routes across the OpenStreetMap network, inferencing three separate XGBoost ensembles to predict multi-horizon temporal variations with incident penalty constraints.

Request (JSON)

json 
{
  "source": {
    "lat": 12.9716,
    "lon": 77.5946
  },
  "destination": {
    "lat": 12.9304,
    "lon": 77.6784
  },
  "departure_time": "2026-05-14T08:30:00Z"
}

Response (200 OK)

json 
{
  "routes": [
    {
      "route_id": 1,
      "label": "Fastest Route",
      "eta_minutes": {
        "1_hour": { "estimate": 34.2, "lower_bound": 30.1, "upper_bound": 41.3 },
        "2_hour": { ... },
        "4_hour": { ... }
      },
      "confidence": { "1_hour": 0.88, "2_hour": 0.76, "4_hour": 0.69 },
      "meta": {
        "distance_km": 14.2,
        "segments": 78,
        "incident": true,
        "incident_segments": 3,
        "route_geometry": [ ... ]
      }
    }
  ]
}
